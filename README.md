# IBM-cognos-analytics

📦 QuickCart Analytics using IBM Cognos Analytics

E-commerce analytics project using IBM Cognos including reports, dashboards, explorations, and storytelling (Q1–Q6).

This project demonstrates online shopping data analysis using IBM Cognos Analytics.

🔹 Q3 – Order Trends

Monthly Order Trends

Dynamic Chart (Bar / Line)

Top 5 Product Categories by Revenue

🔹 Q4 – Deep Exploration

Return Behavior Analysis

Category vs Return Rate

Payment Mode vs Cart Value Insights

🔹 Q5 – Executive Dashboard

Bar, Line, Pie Visualizations

Order Trends and Revenue Distribution
